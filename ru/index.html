Shell Script

Useful Features

  1. 对赋值变量时，需要不留空格

  2. 脚本中的代码每一行或分号隔开均当作单独的命令执行（控制语句除外）

    这导致一种很常见的用法即. /source_path/file 直接运行文件

  3. script是运行时检查的，因此即使存在语法错误，script文件也可能是可以运行的，并且不会在错误时停止

  4. #!/bin/sh通常指定脚本所用shell名称，类似的有#!/bin/bash…，该行也叫shebang行
